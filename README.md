이 어플리케이션을 사용하기 위해선

* " npm install "을 입력하여 프론트엔드 종속성을 다운받아주세요.
* " npm run start "을 입력하여 실행하시면 됩니다.

## Install

```sh
npm install
```

## Run tests

```sh
npm run start
```
